# 🚀 JS-Mini-Projects

A collection of interactive web applications written in pure JavaScript (Vanilla JS). The projects are designed to practice DOM manipulation, event handling, and programming logic.

## 📁 Repository composition

 1. 🧠 English Grammar Quiz

    Interactive quiz to test your knowledge of the English language.
    * Functional
        * Progress indication (Progress Bar)
        * Instant feedback: highlighting correct and incorrect answers
        * Final results screen with a personal message depending on the score.
    * Technologies: JavaScript (objects, arrays, setTimeout), HTML5, CSS3.
2. 🎨 Color Palette Generator

    A tool for designers and developers that allows you to instantly create color schemes.
    * Functional
        * Generate random HEX codes with one click
        * Visual preview of a 5-color palette.
        * Copy to Clipboard function: copies the color code to the clipboard when you click on the icon or the color itself.
        * Copy confirmation animation (icon changes to a check mark).
    * Technologies: JavaScript (random string generation, Clipboard API), Font Awesome for icons.
